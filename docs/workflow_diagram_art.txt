RUNTIME TOOL EXECUTION

ToolExecutor.execute(tool_call)
  1. emit tool_start
  2. WorkflowGuard checks workspace preflight
  3. WorkflowGuard blocks edit_file / existing-file write_file
  4. ApprovalService builds ApprovalRequest with diff_preview
  5. ApprovalService asks user or silent auto-approves
  6. ToolRegistry resolves handler
  7. runtime.run_tool executes with timeout/retry
  8. WorkflowGuard updates needs_verify / preflight state
  9. emit tool_result diff preview when a write succeeds
  10. emit tool_end
  11. return ToolMessage


TASK STATE CONTRACT

TodoManager stores:
  - todo items
  - user_goal
  - constraints
  - files_inspected
  - files_modified
  - decisions
  - test_results
  - open_risks
  - next_steps

Task guard prevents END until:
  - Task State has been created
  - every todo item is completed


SUBAGENT

SubagentRunner keeps isolated child sessions:
  - explorer
  - architect
  - worker
  - tester
  - security

Subagents cannot call subagent or todo, so they cannot recursively delegate or
mutate the parent Task State directly.
