# 三角洲塔防 — 游戏设计文档

> 文件: `examples/html5_canvas_tower_defense/index.html`
> 技术栈: HTML5 Canvas + 纯 JavaScript（单文件，无框架依赖）

---

## 一、核心架构

| 项目 | 详情 |
|------|------|
| 画布尺寸 | 1200 × 700 像素 |
| 网格系统 | 40×40px 方格，30 列 × 17.5 行 |
| 帧率 | requestAnimationFrame，~60fps |
| 音效 | Web Audio API 程序生成（6 种音效） |
| UI 布局 | 顶部状态栏 + 中部画布 + 底部控制栏 |

### 音效类型

| 音效 | 触发时机 |
|------|----------|
| `place` | 放置/铲除防御塔 |
| `shoot` | 防御塔发射子弹 |
| `hit` | 子弹命中敌人（未击杀） |
| `enemyDie` | 击杀敌人 |
| `waveStart` | 新一波开始 / 进入新关卡 |
| `gameOver` | 游戏结束 / 通关 |

## 二、游戏规则

### 2.1 胜利与失败

| 结果 | 条件 |
|------|------|
| **失败** | 生命值降至 0（敌人到达路径终点扣除 1 点生命） |
| **胜利** | 连续通关 5 关（每关存活 20 波，共 100 波） |

### 2.2 核心循环

```
选择防御塔 → 点击地图放置（消耗金币）→ 敌人沿路径前进
→ 塔自动攻击射程内最近敌人 → 击杀获取金币 → 活过 20 波 → 进入下一关
```

### 2.3 操作方式

| 操作 | 说明 |
|------|------|
| **左键点击** | 先选塔类型（底部按钮），再点地图放置 |
| **右键点击** | 铲除已有塔，返还 50% 建造费用 |
| **暂停/继续** | 点击"开始游戏"按钮切换 |
| **速度调节** | ×1 / ×2 / ×3 三档变速 |
| **悬停** | 鼠标悬停已放置的塔可显示射程范围和光圈高亮 |

### 2.4 放置限制

- 不能放在敌人路径上
- 不能与其他防御塔重叠
- 不能超出画布边界
- 金币不足时无法放置（按钮置灰）

### 2.5 战斗规则

| 规则 | 说明 |
|------|------|
| 寻敌 | 每个塔冷却完成后，自动寻找**射程内最近的敌人**开火 |
| 子弹 | 以恒定速度飞行，命中后计算伤害 |
| 溅射伤害 | 以命中点为圆心，范围内的**所有敌人**都受到等量伤害 |
| 冰冻减速 | 命中后目标速度 ×0.5，持续 2 秒 |
| 多目标 | 震荡炮冷却完成后，**同时向范围内所有敌人**发射子弹 |
| 塔间独立 | 每个塔独立计时，互不影响 |

### 2.6 经济系统

| 收入来源 | 金额 |
|----------|------|
| 击杀普通兵 | +12G |
| 击杀疾速兵 | +15G |
| 击杀重装兵 | +25G |
| 击杀 Boss | +100G |
| 铲除已有塔 | 返还 50% 建造成本 |
| 通关进入下一关 | 重置为 500G |

> 无利息、无波次奖励金币——金币完全来自杀敌和铲塔。

## 三、关卡设计

### 3.1 总体结构

```
5 关 × 20 波 = 共 100 波
关间自动继承：通关后进入下一关，金币重置为 500，所有塔清空
```

### 3.2 五关详情

| 关卡 | 名称 | 初始金币 | 初始生命 | 路径折点数 | 路线特点 |
|------|------|----------|----------|------------|----------|
| 1 | 边境巡逻 | 300 | 30 | 6 | 简单 S 弯 |
| 2 | 峡谷通道 | 300 | 25 | 8 | 锯齿折返 |
| 3 | 十字路口 | 300 | 20 | 8 | 高低起伏 |
| 4 | 蛇形回廊 | 350 | 20 | 14 | 密集回形蛇形 |
| 5 | 最后堡垒 | 350 | 15 | 16 | 极曲折长路径 |

#### 路径坐标

**第 1 关 · 边境巡逻**
```
(0,200) → (250,200) → (250,420) → (650,420) → (650,120) → (1200,120)
```

**第 2 关 · 峡谷通道**
```
(0,80) → (280,80) → (280,380) → (560,380) → (560,620)
→ (880,620) → (880,140) → (1200,140)
```

**第 3 关 · 十字路口**
```
(0,320) → (320,320) → (320,80) → (640,80) → (640,520)
→ (960,520) → (960,320) → (1200,320)
```

**第 4 关 · 蛇形回廊**
```
(0,60) → (320,60) → (320,280) → (120,280) → (120,500)
→ (520,500) → (520,160) → (320,160) → (320,380) → (720,380)
→ (720,60) → (920,60) → (920,580) → (1200,580)
```

**第 5 关 · 最后堡垒**
```
(0,80) → (200,80) → (200,620) → (520,620) → (520,160)
→ (340,160) → (340,460) → (700,460) → (700,240) → (520,240)
→ (520,380) → (880,380) → (880,80) → (1060,80) → (1060,520) → (1200,520)
```

### 3.3 难度递进设计

| 参数 | 第1关 | 第2关 | 第3关 | 第4关 | 第5关 |
|------|-------|-------|-------|-------|-------|
| 初始金币 | 300 | 300 | 300 | 350 | 350 |
| 初始生命 | 30 | 25 | 20 | 20 | 15 |
| 路径复杂度 | 低 | 中 | 中 | 高 | 极高 |

→ 难度递增靠三条腿：生命越来越少，路径越来越曲折，留给玩家的布阵空间越来越受限。

### 3.4 波次递进规则

| 波次段 | 敌人构成 |
|--------|----------|
| 1–4 波 | 仅普通兵 |
| 5–9 波 | 普通兵 + 疾速兵（50/50 随机） |
| 10–14 波 | 重装兵(60%) + 普通/疾速混合 |
| 15–19 波 | 全部类型随机混出（不含 Boss） |
| **第 20 波** | 敌人数 ×10，**最后 20 个全是 Boss** |

**每波敌人数公式**：
```
敌人数 = 4 + floor(波次 × 1.3)
第 20 波敌人数 = 以上 × 10
```

> 例：第 20 波基础敌人数 ≈ 4 + 26 = 30，×10 = **300 个敌人**，其中最后 20 个是 Boss。

**生成间隔公式**：
```
生成间隔 = max(700, 1600 - 波次 × 50) ms
```

> 波次越高出兵越快，从 1600ms 逐步降至最低 700ms。

## 五、敌人

### 5.1 敌人类型

| 敌人 | 生命 | 速度 | 击杀奖励 | 体型 | 首次出现 |
|------|------|------|----------|------|----------|
| **普通兵** | 45 | 1.5 | 12G | 16px | 第 1 波 |
| **疾速兵** | 40 | 3.0 | 15G | 14px | 第 5 波 |
| **重装兵** | 200 | 0.8 | 25G | 22px | 第 10 波 |
| **Boss** | 800 | 0.5 | 100G | 32px | 第 20 波（最后20个） |

### 5.2 美术风格

| 敌人 | 视觉特征 |
|------|----------|
| 普通兵 | 8 角星形 + 白色眼球 + 黑色瞳孔 + 红色描边 |
| 疾速兵 | 金色椭圆 + 白色速度条纹 + 金色眼睛 |
| 重装兵 | 紫色圆形装甲 + 红色怒眼 + 紫色护额横条 |
| Boss | 暗红脉冲光环 + 暗黑盔甲 + 恶魔双角 + 猩红发光眼 + 金色瞳孔 + 白色獠牙 |

### 5.3 血条

- 宽度 = 体型 × 1.5
- 颜色动态变化：绿色(>50%) → 黄色(25~50%) → 红色(<25%)

## 八、设计意图总结

| 设计维度 | 手法 |
|----------|------|
| **前期教学** | 第 1 关路径简单、生命充裕(30)、仅出普通兵 |
| **中期挑战** | 路径变复杂，疾速兵和重装兵引入，需混合布塔 |
| **后期高压** | 生命紧张(15)、Boss 海(20只×800血)、出兵极快(700ms) |
| **终局检验** | 第 5 关 16 折点长路径 + 仅 15 生命 + Boss 狂潮 |
| **策略深度** | 冰冻控场 + 溅射清群 + 狙击点杀 + 震荡炮多目标，塔阵组合是关键 |
