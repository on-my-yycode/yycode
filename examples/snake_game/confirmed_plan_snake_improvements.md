## 改进项

### 1. 暂停功能 ⏸️
- 按 `P` 或 `Space` 暂停/继续
- 暂停时绘制半透明遮罩 + "游戏暂停" 文字
- 新增 `self.paused` 状态

### 2. WASD 键盘支持 ⌨️
- 补充 `W/A/S/D` 方向键（README 已声称支持但代码未实现）
- 与方向键逻辑一致，不可反向

### 3. 最高分记录 🏆
- 用本地文件 `snake_highscore.txt` 保存最高分
- 游戏启动时加载，结束时若超过则保存
- 在分数旁显示 "最高分: xxx"

### 4. 难度递增 ⚡
- 初始 FPS=8，每吃 5 个食物 +2 FPS
- 最高 FPS=20
- 提升挑战性

### 5. 音效 🔊
- 使用 `pygame.mixer.Sound` 生成简单波形音效
- 吃食物：短促高频音效
- 游戏结束：低沉提示音
- 无需外部音频文件

### 6. 食物脉冲动画 ✨
- 食物大小随 sin 时间周期性变化
- 在现有圆形食物基础上增加动态效果

### 7. 蛇初始长度改为 3 🐍
- `snake.py` 初始蛇长从 1 改为 3
- 更符合贪吃蛇经典体验

### 8. 修复 README 与代码不一致 📝
- README 声称支持 P 暂停、R 重开、ESC 退出、WASD → 全部实现

### 9. 修复测试与代码不匹配 🧪
- `test_initial_snake_length` 期望 3，代码改为 3 后通过
- 补充暂停、WASD 相关测试
