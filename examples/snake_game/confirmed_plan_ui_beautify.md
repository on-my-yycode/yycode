# 确认计划：贪吃蛇 Python 版 UI 美化

## 用户目标
美化 `examples/snake_game/snake.py` 的图形界面，提升视觉体验。

## 确认范围
- **仅美化 Python pygame 版** (`snake.py`)
- **HTML5 版不动**

## 非目标
- 不改变游戏核心逻辑（移动、碰撞、分数等）
- 不引入新依赖（仅使用 pygame 内置能力）
- 不修改 HTML5 版

## 美化项目

### 1. 渐变深色背景 + 网格线
- 用深蓝到黑的垂直渐变替代纯黑背景
- 绘制淡色细网格线增强空间感

### 2. 蛇身美化
- 每节身体用圆角矩形
- 蛇头更深色调 + 白色眼睛（根据方向定位）

### 3. 食物美化
- 圆形食物 + 脉冲光晕动画（用 sin 时间函数控制）
- 外围发光效果

### 4. 粒子特效
- 吃到食物时在食物位置生成若干粒子
- 粒子向外扩散、逐渐消失

### 5. 分数面板
- 半透明深色背景卡片
- 分数文字居中显示

### 6. 游戏结束遮罩
- 半透明黑色遮罩覆盖全屏
- 大字标题 + 提示文字

## 实现计划
单文件修改 `snake.py`，在现有 `SnakeGame` 类中增加绘制方法。

## 验证
- 运行 `python snake.py` 确认游戏正常启动
- 视觉确认各美化项生效
- 确认方向控制、碰撞、计分逻辑不受影响
- 运行 `python -m pytest test_snake.py -v` 确保 8 个测试全通过
